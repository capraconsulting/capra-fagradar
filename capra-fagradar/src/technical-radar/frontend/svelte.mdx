Svelte er et moderne rammeverk for utvikling av brukergrensesnitt i JavaScript.
Det skiller seg fra andre populære rammeverk som React og Vue ved å kompilere applikasjonskoden til JavaScript i byggeprosessen,
i stedet for å bruke en runtime i nettleseren. Dette gjør applikasjonene lettere, raskere og mer effektiv.
