Web Components er en teknologi som lar utviklere lage gjenbrukbare, egendefinerte elementer for nettet ved hjelp av standardiserte API-er.
Disse komponentene fungerer uavhengig av hvilket rammeverk eller bibliotek som brukes, fordi de er basert på nettets innebygde funksjonalitet.

Web Components gir utviklere muligheten til å lage tilpassede HTML-elementer som oppfører seg som native nettleserelementer.
