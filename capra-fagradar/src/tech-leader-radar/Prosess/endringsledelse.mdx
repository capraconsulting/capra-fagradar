# Hva er det?
Endringsledelse handler om å systematisk håndtere organisatoriske endringer - fra strategi og teknologi til arbeidsprosesser. Det omfatter planlegging, gjennomføring og forankring av endringer med fokus på menneskene som påvirkes.

# Hvorfor er det viktig for oss?
Som teknologiledere leverer vi løsninger som endrer hvordan folk jobber. God endringsledelse øker sjansen for vellykket implementering og varig gevinstrealisering.

# Hva gjør vi hvis vi vil bli bedre på dette?
* Delta på sertifiseringer innen anerkjente rammeverk som PROSCI eller ADKAR, og bruk metodikken aktivt i prosjekter for å bygge praktisk erfaring

* Utvikle kompetanse i organisasjonspsykologi og kommunikasjon gjennom kurs og faglitteratur - særlig rundt motstand mot endring og hvordan skape eierskap

* Dokumenter erfaringer fra vellykkede og utfordrende endringsprosesser, og del denne kunnskapen systematisk i fagmiljøet gjennom samlinger og diskusjoner
