# Hva er det? 
Feedback er tilbakemeldinger som hjelper enkeltpersoner eller team med å forstå hvordan deres handlinger påvirker andre, og hvordan de kan forbedre seg.
Det kan være både positiv (forsterkende) og konstruktiv (utviklende) tilbakemelding.
Effektiv feedback er konkret, rettidig og handler om observerbare handlinger, ikke personlige egenskaper.
